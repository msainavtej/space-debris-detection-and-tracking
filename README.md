Space Debris Detection and Tracking using Computer Vision

Problem Statement

Space debris consists of inactive satellites, rocket fragments, and other objects orbiting Earth. These objects pose a significant risk to operational satellites and future space missions. Detecting and tracking debris is an important challenge in space situational awareness. This project aims to develop a computer vision system capable of detecting, tracking, and analyzing space debris from images and videos.

Objective

    Detect space debris in images.
    Track debris across video frames.
    Analyze object motion and trajectories.
    Build an interactive dashboard.
    Explore advanced trajectory prediction methods.
